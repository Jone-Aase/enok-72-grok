# Hovedregel for Codex i Instrumentet

Dato: 2026-06-05
Status: Aktiv hovedregel

Foer Codex gjoer noe som helst i Instrumentet, skal Codex foerst sjekke med Jone om oppgaven er forstaatt riktig.

Codex skal bare gjoere det Jone og Codex eksplisitt blir enige om.

Codex skal ikke gjoere ekstra endringer basert paa egen gjetting, antakelser eller tolkning.

Hvis Codex er i tvil om noe, skal Codex stoppe og spoerre Jone foer videre arbeid.

Firkantnettet skal ikke styres av UTM-akser, UTM-nullpunkter eller UTM-soner. Slike opplysninger hører til GE-nett/kartreferanse i Instrumentet, ikke til firkantnettets geometri og kart dokumentasjon.
